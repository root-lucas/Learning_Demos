## Usage

`/lib/`目录下有很多`main.xx.dart`文件，每一个文件就是一个小demo，想看不同的UI demo只需复制其他的文件粘贴至`main.dart`即可。

```sh
flutter packages get  # 获取依赖
flutter run 		 # 运行
```
